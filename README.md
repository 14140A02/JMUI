# JMUI
    
    哇咔咔我是吊炸天的介绍.

## 目录结构

- index.html
    + 示例入口文件
- demo/
    + 示例 
- stylus/
    + 存放 .styl 源码
- css/
    + 执行 `$ grunt stylus` 后生成的 *.css 文件 
- js/
    + 存放 .js 源码
- lib/
    + 存放第三方依赖


## 代码规范

代码请遵命[AlloyTeam 团队规范](http://alloyteam.github.io/code-guide/#css)

## 构建方式

    $ git clone https://github.com/AlloyTeam/JMUI.git

    $ cd JMUI

    $ npm install 

    $ grunt
